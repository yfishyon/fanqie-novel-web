# fanqie-novel-web
番茄小说下载的PHP实现，采用API模式
## 特点
十分方便的下载，支持资源共享
一个纸张写的代码，引用了simple_html_dom.php
## 使用方法
上传服务器，访问域名/dl-book.php?id=书本id
稍等亿会，即可下载完成
